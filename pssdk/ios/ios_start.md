# <span id="jump1">1.工程引入</span>

1）[下载](https://github.com/Avid-ly/iOS-PSSDK-Demo/releases) `PSSDK`

2）将 `PSSDK.framework`和`PSSDK.bundle`导入工程中

3）引入 `PSSDK`

```
#import &lt;PSSDK/PSSDK.h>
```
